# That Wall - A posting board developed by Kyle Williams

## Features: 
- SPA
- Confirmation Emails Powered by SendinBlue SMTP
- User Authentication with JWT
- RESTful API

## Technologies: 
- React.js
- Express.js
- React Hooks
- Context API
- PostgreSQL & Sequelize
- JWT
- Axios
- Argon2
- Jest
- Supertest
